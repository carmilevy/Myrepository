GitHub Pull Requests

What is a Pull Request?

A pull request (PR) lets you tell others about changes you've pushed to a branch in a repository on GitHub. Once a pull request is opened, you can discuss and review the potential changes with collaborators and add follow-up commits before your changes are merged into the base branch.

Creating a Pull Request

  1. Navigate to the repository on GitHub.
  2. Click on the "Pull requests" tab.
  3. Click the "New pull request" button.
  4. Select the branch you want to merge.
  5. Fill in the details and click "Create pull request".

Benefits of Pull Requests

  • Code review and feedback
  • Collaboration
  • Documentation of changes
